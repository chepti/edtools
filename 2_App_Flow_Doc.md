# App Flow

## 1. Landing Page (Homepage)
- Header: לוגו, ניווט, טוגל מצב כהה/בהיר, כניסה/הרשמה
- רצועת כלים מומלצים / חדשים / טרנדיים
- רצועת מדפים בולטים (כמו "חינוך מיוחד", "שילוב AI באנגלית")
- חיפוש מהיר
- כפתור CTA: "מצא כלי שמתאים לך עכשיו"

## 2. Tools Directory (דף כל הכלים)
- סינון לפי צ'יפים (נושאים, רמות, שימושים וכו')
- הצגה ככרטיסים: שם, לוגו, דירוג, כמות מדפים/דוגמאות/הדרכות
- לחיצה על כרטיס → דף כלי

## 3. Tool Detail Page
- תיאור מלא
- יתרונות, חסרונות, מגבלות
- טכנולוגיה, שפת תמיכה, רמת קושי, סוג תוצר
- דירוג הכלי, סקירות של משתמשים
- רצועות:
  - דוגמאות (גלריית קלפים)
  - הדרכות (רשימה או גלריה)
  - מדפים שבהם מופיע הכלי
- כפתור לדיווח בעיה/תוכן לא הולם

## 4. Shelf (מדף כלים)
- כותרת, תיאור
- מי יצר? (כינוי/מוסד)
- רשימת כלים + תצוגה מקדימה
- אפשר לעקוב/לשמור
- כל מדף ציבורי

## 5. Add Example / Add Tutorial
- טופס קליל:
  - 📝 שם, 📎 קישור, 🧾 תיאור, 🙋‍♀️ קרדיט, 🧑‍🏫 מי תרם
  - ⭐ דירוג אופציונלי
- אימוג'ים + טולטיפים להכוונה
- שליחה מיידית, מופיע בטיוטה עד אישור

## 6. Create Shelf
- טופס יצירה
- בוחר כלים ע"י חיפוש או גלילה
- מגדיר כינוי מוצג
- שמירה → מדף ציבורי

## 7. Admin Panel
- הוספת כלי חדש
- אישור/מחיקה של תכנים שסומנו ע"י משתמשים
- ניהול תוכן ודיווחים
