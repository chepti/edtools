# App Functionality

## User Types
- 🧑‍🏫 Teacher (רגיל)
  - יכול לדפדף, לסנן ולצפות בכלים
  - יכול להוסיף דוגמאות, הדרכות, מדפים
  - יכול לדרג כלים, הדרכות, ודוגמאות
  - יכול לדווח על תכנים
  - יכול לבחור להציג כינוי בלבד

- 🛠️ Admin
  - כל פעולות המורה +
  - הוספת כלים
  - ניהול תכנים מדווחים
  - עריכה ומחיקה

## Functionalities

### 🔍 Browsing & Search
- סינון צ'יפים: נושא, שימוש פדגוגי, חינמיות, רמת קושי, עברית
- חיפוש חופשי בשם הכלי

### 📦 Tools
- כרטיסים עם תצוגה מקדימה
- עמוד כלי עם פרטים מלאים
- הצגת קשרים (הדרכות, דוגמאות, מדפים)

### ➕ Adding Content
- טפסים נעימים ואינפורמטיביים
- אימוג'ים + טולטיפים
- הצגה בטיוטה עד אישור (לפי צורך)

### 📚 Shelves
- כל מדף ציבורי
- יכול להיווצר ע"י כל מורה
- ניתן לעקוב אחרי מדף

### 🛡️ Moderation
- כפתור "דווח" זמין בכל כרטיס או תוכן
- הדיווח נשלח לפאנל האדמין
- רק אדמין מאשר תכנים חדשים (אם נדרש)

### 🌗 UI/UX
- מצב כהה/בהיר
- עיצוב רספונסיבי
- ממשק נעים ומעודד תרומה
